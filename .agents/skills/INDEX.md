# .agents/skills INDEX

This index lists installed skills for quick discovery. Each entry should include:
- name
- version
- tags
- source
- short-desc
- installed-at (YYYY-MM-DD)
- last-used (YYYY-MM-DD)

Agent will maintain this file when installing or removing skills.

Example entry:

- name: fix-deps
  version: 0.1.2
  tags: [ci, fix]
  source: https://github.com/example/fix-deps
  short-desc: "Fix common dependency issues for Node projects"
  installed-at: 2026-04-27
  last-used: 2026-04-27
